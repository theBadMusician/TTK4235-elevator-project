double timer_getWallTime ( void  )

Retrieves the current system wall time in seconds.

  • Returns
    The current time in seconds as a double-precision float.
  • Fetches the time using gettimeofday. The seconds and microseconds are combined into a single value. The mathematical conversion used is $t_{sec} = s + \mu s \times 10^{-6}$ .

◆ timer_isTimeout()

bool timer_isTimeout ( timerAlarm *  timerAlarmInstance)

Checks if an active timer has expired.

  • Parameters
    [in]timerAlarmInstancePointer to the timer instance to check.
  • Returns
    true if the timer is active AND the current wall time has passed the end time.
    false if the timer is inactive or the time has not yet expired.
  • This is a non-blocking check. It relies on polling the current wall time against the stored expiration time.

Starts a timer for a specified duration.

Parameters
[in,out]timerAlarmInstancePointer to the timer instance.
[in]durationThe amount of time to run the timer for, in seconds.

Calculates the absolute end time based on the current wall time plus the requested duration. If the timer is already active, this function returns immediately without overwriting the running timer.
