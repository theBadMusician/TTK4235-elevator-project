5 * This module implements the core elevator algorithm (the LOOK algorithm).

8 * between floors.

38/**
39 * @brief Determines the initial direction if the elevator is halted between floors.
40 * * @param[in,out] elev Pointer to the central elevator state structure.
41 * @return true if a recovery was necessary and handled.
42 * @return false if the elevator is not in a recovery state.
43 * * @details If the elevator is stopped between floors (sensor reads -1) and a new

50/**
51 * @brief Evaluates whether the elevator should halt at the current floor.
52 * * @param[in] elev Pointer to the central elevator state structure.
53 * @return true if the elevator should stop at this floor.
54 * @return false if the elevator should bypass this floor.
55 * * @details Implements the core stopping logic: Always stops for cabin calls.
56 * If moving UP, it stops for UP hall calls, or DOWN hall calls if it is the
57 * highest active request. If moving DOWN, it stops for DOWN hall calls, or
58 * UP hall calls if it is the lowest active request.
59 */
60bool movement_stopHandler(ElevatorState* elev);
61
62/**
63 * @brief Calculates and applies the next motor direction based on active orders.
64 * * @param[in,out] elev Pointer to the central elevator state structure.
65 * * @details Checks for remaining requests in the current direction of travel.
66 * If orders remain, it continues in that direction. If the path is clear, it
67 * reverses direction to serve orders behind it. If the entire queue is empty,
68 * it stops the motor and transitions the system into STATE_IDLE.
69 */
70void movement_directionHandler(ElevatorState* elev);
