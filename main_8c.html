int main ( )

Initializes the system and runs the main FSM loop.

  • The function kicks off the elevator control system and enters an infinite loop, calling fsm_spin() to evaluate inputs and update the state machine. To prevent the program from consuming 100% of the CPU, the loop yields execution for 20 milliseconds between each spin, resulting in an effective system polling rate of 50 Hz.
  • Returns
    int Returns 0 upon successful execution (in practice, this loop runs infinitely until the process is manually terminated).
