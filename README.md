# TTK4235-elevator-project

This project involves the development of a real-time control system for a physical elevator model. The software is implemented in C and interfaces with the hardware via an Arduino microcontroller. The primary objective is to create a robust, event-driven system capable of safely managing concurrent user requests across multiple floors while adhering to real-time performance constraints.

To ensure a structured development lifecycle and verification of the system's acceptance criteria, the project followed the pragmatic V-model. This methodology, combined with uml design principles, allowed the complex control logic to be decomposed into independent modules, centered around fsm and an elevator-routing logic.
